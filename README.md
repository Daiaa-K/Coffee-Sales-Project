# Coffee Sales Project

A sales dataset tracking coffee orders, customers, and products across three markets: the United States, Ireland, and the United Kingdom.

## File Overview

**File:** `Coffee_Sales_Project.xlsx`

The workbook contains 6 sheets covering raw transaction data, customer and product reference tables, pivot summaries, and a dashboard.

## Sheets

| Sheet | Description |
|---|---|
| `orders` | Individual order records including order ID, date, customer, product, quantity, and sales amount |
| `customers` | Customer reference table with contact details, location, and loyalty card status |
| `products` | Product catalog with coffee type, roast type, size, unit price, and profit margin |
| `Total Sales` | Pivot table of total sales by year and month, broken down by coffee type |
| `Sales by Country` | Pivot summary of total sales per country |
| `top 5 customers` | Ranked list of the five highest-spending customers |
| `Dashboard` | Visual summary (charts/graphs built in Excel) |

## Key Details

- **Date range:** 2019–2022
- **Coffee types:** Arabica, Excelsa, Liberica, Robusta
- **Roast types:** Light, Medium, Dark
- **Sizes:** 0.2 kg, 0.5 kg, 1 kg, 2.5 kg
- **Markets:** United States, Ireland, United Kingdom
- **Total revenue (all years):** ~$45,382
